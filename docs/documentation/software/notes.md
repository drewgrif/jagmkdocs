# Notes

The objection about syncing can be eliminated if you are using Nextcloud Desktop OR 
A Git repo for notes that you manually sync.

---

### **1. Geany with Markdown Plugin: 8.5**
**Pros:**
- **Lightweight and Fast:** Geany is a small, fast IDE that works well even on older systems.
- **Extensive Language Support:** While it’s primarily for code, the markdown plugin adds support for markdown editing with syntax highlighting and preview.
- **Cross-platform:** Available on Windows, Linux, and macOS.
- **Highly Customizable:** You can add various plugins and configure settings to suit your needs.
- **Free and Open-source:** Fully free without any premium features.

**Cons:**
- **Not Designed for Note-Taking:** Geany is a general-purpose text editor, not specifically for note-taking, so it lacks features like task management or note linking.
- **Limited Markdown Features:** While functional, markdown support is basic compared to dedicated markdown apps.
- **UI Can Be Too Minimalistic:** Some users may find the interface too bare-bones or difficult to navigate.

---

### **2. VSCode (vscodium) with Plugins: 8.1**
**Pros:**
- **Highly Customizable:** With a plethora of plugins, you can tailor it to your specific needs for markdown editing.
- **Powerful Editor:** Features like code autocompletion, debugging, and Git integration are useful if you're a developer.
- **Markdown Preview:** Built-in markdown preview with live update.
- **Cross-platform:** Available on Windows, macOS, and Linux.
- **Advanced Features:** Excellent for those needing more advanced text editing tools.

**Cons:**
- **Heavyweight:** VSCode can feel bloated if you're only using it for markdown editing.
- **Complex Setup:** Requires setup of plugins and configuration, which may take time.
- **Not Specialized for Notes:** It lacks features specifically tailored to note-taking, like task management or linking.
- **UI Complexity:** The interface may be overwhelming for those who prefer a simpler, more focused writing environment.

---

### **3. Iotas: 7.9** (installed using Flathub)
**Pros:**
- **Modern Interface:** Iotas is designed with simplicity in mind, offering an intuitive, easy-to-use interface.
- **Markdown Support:** Full support for markdown with a live preview and rich formatting options.
- **Focus on Organization:** Great for organizing and structuring notes with hierarchical elements.
- **Cross-Platform:** Available on Windows, macOS, and Linux, with a consistent experience across platforms.
- **Lightweight:** It’s not a resource-heavy application, so it works well on low-spec systems.

**Cons:**
- **Limited Features:** Lacks advanced features like graph view or backlinks that are available in more specialized note-taking apps like Obsidian or Logseq.
- **Syncing Issues:** Lacks a built-in sync feature, so you’ll need to use third-party tools like cloud services for syncing across devices.
- **Niche Tool:** While it’s good for basic markdown editing and note organization, it doesn’t support advanced use cases like task management or complex note interlinking.

---

### **4. Zettlr: 7.9**
**Pros:**
- **Markdown and Citation Support:** Zettlr is great for academic writing with built-in citation management and markdown support.
- **Cross-platform:** Works on Windows, macOS, and Linux.
- **Multiple File Formats:** Can export to various formats like PDF, DOCX, and HTML.
- **Open-source:** Free and customizable.
- **Dark Mode:** Great for long writing sessions with an eye-friendly dark theme.

**Cons:**
- **Complex UI:** The interface can feel overwhelming, especially for users who prefer simplicity.
- **No Cloud Sync:** Does not have built-in cloud syncing; you’ll need to rely on external services like Dropbox.
- **Limited Collaboration Features:** Doesn’t have built-in support for real-time collaboration.

---

### **5. Apostrophe: 7.5**
**Pros:**
- **Focused on Writers:** Designed for writers with a clean and distraction-free interface.
- **Markdown and Live Preview:** Full markdown support with a real-time preview.
- **Focus Mode:** Great feature for distraction-free writing.
- **Cross-platform:** Works on Windows, macOS, and Linux.
- **Simple UI:** Minimalistic interface makes it easy to use for quick note-taking.

**Cons:**
- **Basic Features:** Lacks more advanced note-taking tools like task management or note linking.
- **Limited Export Options:** Exporting notes is somewhat restricted compared to other editors.
- **Not Specialized for Note-Taking:** While good for writing, it doesn't offer as much for knowledge management.

---

### **6. Obsidian: 7.8**
**Pros:**
- **Local Storage:** Stores notes locally for full control over your data.
- **Linking & Graph View:** Allows for powerful interlinking between notes and a graph view for visualizing connections.
- **Customizable Plugins:** A wide range of plugins to extend functionality, from daily notes to task management.
- **Cross-platform:** Available on Windows, macOS, Linux, and mobile (iOS/Android).
- **Markdown Support:** Real-time preview of markdown and excellent formatting options.

**Cons:**
- **Learning Curve:** New users may find the interface overwhelming, especially with its many options and plugins.
- **No Built-in Syncing:** Syncing requires third-party services or a paid plan for Obsidian Sync.
- **Resource Heavy:** Can become slow with large vaults or too many plugins.
- **No WYSIWYG Editing:** The lack of a What You See Is What You Get (WYSIWYG) editor can be a downside for users who want a more polished writing experience.

---

### **7. MarkText: 7.2** - Not being maintained
**Pros:**
- **Simple and Elegant UI:** Offers a clean and user-friendly interface.
- **Live Preview:** Provides a split-screen live preview to see both raw markdown and formatted content.
- **Cross-platform:** Available on Windows, macOS, and Linux.
- **Open-Source:** Free and open-source with no hidden restrictions.

**Cons:**
- **Limited Features:** Lacks advanced features like backlinks or graph view.
- **No Cloud Sync:** Syncing requires manual file management or third-party services.
- **Basic Markdown Support:** Doesn't support advanced markdown features like LaTeX or complex table handling.
- **Occasional Bugs:** Some users report occasional crashes or bugs in the newer versions.

---

### **8. Joplin: 6.5**
**Pros:**
- **Open-source:** Free with no restrictions.
- **Cross-platform:** Available on Windows, macOS, Linux, Android, and iOS.
- **End-to-End Encryption:** Syncing across devices with end-to-end encryption for security.
- **Web Clipper:** Clip web content into your notes directly.
- **Markdown Support:** Supports markdown formatting with a built-in editor and preview.

**Cons:**
- **Outdated Interface:** Some users find the UI clunky and not as polished as other apps.
- **Sync Issues:** Syncing across devices can occasionally be slow or unreliable.
- **Limited Features:** Lacks advanced features like backlinks, graph view, or task management.
- **Performance Issues:** Can be sluggish with larger note databases.

---

### **9. QOwnNotes: 5.8**
**Pros:**
- **Open-source and Cross-platform:** Works with Windows, macOS, and Linux.
- **Markdown Support:** Supports markdown syntax with live preview and syntax highlighting.
- **Task Management:** Includes basic task management features like checklists.
- **Sync with Nextcloud:** Seamlessly syncs with Nextcloud for cross-device use.

**Cons:**
- **Outdated UI:** The interface is clunky and could benefit from a modern redesign.
- **Limited Features:** Lacks advanced features like linking or task management beyond basic to-do lists.
- **No Mobile App:** Lacks an official mobile app, limiting its accessibility.
- **Syncing Issues:** Syncing with Nextcloud can sometimes be unreliable.

---

### **10. Logseq: 5**
**Pros:**
- **Outliner-based:** Focuses on hierarchical note-taking, making it great for knowledge management.
- **Markdown & Org Mode:** Supports both markdown and org mode syntax.
- **Local Storage & Syncing Options:** Keeps everything local, but you can set up third-party syncing like GitHub or Dropbox.
- **Open-source:** Free to use and fully customizable.

**Cons:**
- **Clunky UI:** Some users find the interface unintuitive and hard to navigate.
- **Steep Learning Curve:** The outliner-based approach can be challenging for users unfamiliar with this style.
- **Limited Export Options:** Exporting notes is less flexible compared to other editors.
- **Performance Issues:** Slows down with large databases or complex notes.
